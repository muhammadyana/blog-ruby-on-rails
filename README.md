## INSTALL DEPENDENCIES
```bash
$ bundle install
```
## CREATE DATABASE
```bash
$ rake db:create
```
## RUN MIGRATION
```bash
$ rake db:migrate
```

## RUN SERVER
```bash
$ rails s
```
